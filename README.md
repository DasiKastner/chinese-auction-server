מערכת מכירה סינית## – צד שרת (C#)

מערכת מכירה סינית היא מערכת לניהול הגרלות (Chinese Auction), הכוללת ניהול משתמשים, מתנות, תרומות, רכישות והגרלות, עם דגש על מבנה נכון של נתונים ו־API ברור להרחבה עתידית.

### תיאור הפרויקט
צד השרת בנוי ב־C# ומספק API לניהול כל הלוגיקה של המערכת.  
המערכת כוללת את הישויות והתהליכים הבאים:

- משתמשים – הרשמה, התחברות וניהול פרטי משתמש
- מתנות – יצירה, עדכון, מחיקה ושליפה של מתנות
- קטגוריות – סיווג מתנות לפי קטגוריות
- תורמים – ניהול תורמים של מתנות
- רכישות – רכישת כרטיסים להגרלות
- הגרלות – שיוך משתמשים למתנות והשתתפות בהגרלה
- קביעת זוכה – שמירת שם הזוכה עבור כל מתנה

המערכת בנויה בצורה רלציונית עם קשרים ברורים בין הטבלאות ומאפשרת הרחבה קלה לפיצ'רים נוספים.

## מבנה הנתונים (Database)

המערכת מבוססת על SQL Server וכוללת את הטבלאות הבאות:

### User
### Gift
### Category
### Donor
### Purchase
### Lottery

### קשרים מרכזיים:
- משתמש יכול לבצע מספר רכישות
- כל רכישה שייכת למתנה ולמשתמש
- מתנה שייכת לקטגוריה ולתורם
- טבלת Lottery מקשרת בין משתמשים למתנות לצורך הגרלה

## Controllers (API)

הפרויקט כולל את הקונטרולרים הבאים:

- AuthController – הרשמה והתחברות
- User (דרך Auth או הרחבה עתידית)
- CategoryController – ניהול קטגוריות
- GiftController – ניהול מתנות
- DonorController – ניהול תורמים
- PurchaseController – ניהול רכישות
- LotteryController – ניהול הגרלות

## טכנולוגיות

- Backend: C# (.NET / ASP.NET Core)
- Database: SQL Server
- ORM: Entity Framework Core
- ארכיטקטורה: REST API

## תכונות עיקריות

- ניהול משתמשים והרשאות
- ניהול מתנות וקטגוריות
- ניהול תורמים
- ביצוע רכישות (כרטיסים להגרלה)
- השתתפות בהגרלות
- שמירת זוכה לכל מתנה
- מבנה נתונים רלציוני ברור ומאורגן
- הפרדה בין שכבות (Controllers / Data)
